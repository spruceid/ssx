# ssx-test-dapp

This is a dapp to test SSX functionality.


**Install Dependencies**

```bash
yarn install 

# or 

npm install
```

**Setup Alchemy Account**
This example uses `alchemy-sdk`, to use it you will need an Alchemy API Key.

**Run**

``` bash
yarn start

# or

npm run start
```

**Build**

``` bash
yarn build

# or 

npm run build
```
